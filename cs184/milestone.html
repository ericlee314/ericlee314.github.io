Light Field Rendering for Quick Focal Changes and Stereoscopic 3D

Eric Lee, CS184-ach

Xintian Meng, CS184-adz


Milestone Summary

We have extended our project 3 ray tracer to generate a 4-dimentional light field instead of rendering a single image from a stationary camera. Our ray tracer renders multiple images of the same scene from variable camera positions. This creates a 2-dimentional array of images where each with a slightly different perspectives. By combining these images, we can generate several interesting effects. For instance, we have simulated a depth of field effects with varibale focal distance by averaging the images in Photoshop. We have also created stereoscopic 3D images which allow the user to perceive depth and relative distance when viewed with 3D glasses. Finally, we have equipped our ray tracer to generate a depth map with light values corresponding to surface distance from viewpoint which will assist us in creating an interavtive experience where the user can refocus the sence on any point of clicking.

Progress and Future goals

We have achieved the functionality of shifting the camera position by specific (u, v) offsets to create individual light field images. Following images include the final results of a depth of field effects with short/long focal distance, a steroescopic 3D image and a depth map, and they are created from a 3x3 image array, where each image (400x400) only takes a few seconds to render. (With the exception of the depth map, the images below were compiled in Photoshop from the original light field renders.)



Future goals include: Start building an interactive GUI/live demo system that can quickly generate images with custom focal distance; Begin rendering a larger scale light field images for GUI/live demo; Add in VR support as an extra.
